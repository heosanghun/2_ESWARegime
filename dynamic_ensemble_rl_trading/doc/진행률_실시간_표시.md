# 진행률 실시간 표시

## 1. 한 번만 보기 (현재 진행률 1회 출력)

```bash
python scripts/progress_once.py
```

- `results/verification/progress_reach_100.json` 또는  
  `results/verification/reach_100_autonomous.log` 를 읽어  
  **시도 횟수 / 전체, 진행률(%), 최고 일치성, 상태** 를 출력합니다.

## 2. 실시간 진행률 (주기적 갱신)

```bash
python scripts/show_realtime_progress.py
```

- **5초마다** 진행 파일/로그를 다시 읽어 갱신합니다.
- 진행률 바, 최고 일치성, 현재 blend/scale, 마지막 NEW BEST 로그를 출력합니다.
- `status: done` 이 되거나 10분간 갱신이 없으면 자동 종료됩니다.
- 중단하려면 `Ctrl+C` 입력합니다.

## 3. 자율 실행 (100% 일치 탐색)

```bash
python scripts/reach_100_percent_autonomous.py
```

- 실행 중에 `progress_reach_100.json` 이 갱신되므로  
  **다른 터미널에서** `show_realtime_progress.py` 를 실행하면 실시간 진행률을 볼 수 있습니다.

## 4. 출력 예시

```
=== Reach 100% autonomous progress (from log) ===
[##############################--] 20/20  (100.0%)
Best avg consistency: 31.5%  (blend=0.8 scale=1.0)
Status: done
Log: results/verification/reach_100_autonomous.log
```

실시간 모드:

```
[################################--------] 12/20 (60.0%)
Best consistency: 28.1%  |  blend=0.70 scale=1.30  |  done_try
Updated: 2026-02-13 13:00:48  |  status: running
----------------------------------------------------------------------
```
