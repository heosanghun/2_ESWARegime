# 학습/검증 파이프라인 진행률 확인 방법

## 실시간 진행률 보기

프로젝트 루트에서 다음 명령을 실행하세요.

```bash
cd dynamic_ensemble_rl_trading
python scripts/show_progress.py
```

- **콘솔:** 현재 단계, 단계명, 에이전트 진행 (예: `Agents: 2/15 (13.3%)`)
- **파일:** `results/verification/progress.md` 가 갱신됨 (표 + 진행 바)

## 진행률 파일 위치

- **진행률 요약:** `results/verification/progress.md`
- **전체 로그:** `results/verification/train_and_verify.log`

## 현재 파이프라인 (1M timesteps)

| 단계 | 내용 | 예상 소요 |
|------|------|-----------|
| 1 | Regime Classifier 학습 | 약 5분 |
| 2 | PPO 15개 에이전트 (각 1,000,000 steps) | 약 8시간 (에이전트당 ~32분) |
| 3 | 테스트 기간 백테스트 | 약 5분 |
| 4 | 논문 Table 2 비교 | 즉시 |

**진행률 해석**
- Step 2 진행: `2/15 (13.3%)` = Bull pool 2번째 에이전트 완료, 3번째 학습 중
- 15개 모두 끝나면 STEP 3(백테스트) → STEP 4(비교)가 자동 실행됩니다.

## 주의사항

- 학습이 백그라운드로 실행 중이면 터미널을 닫지 마세요.
- 진행률을 다시 보려면 `python scripts/show_progress.py` 를 다시 실행하면 됩니다.
