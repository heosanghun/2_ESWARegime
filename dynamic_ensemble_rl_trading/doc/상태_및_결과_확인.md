# 상태 및 결과 확인 (한 번에)

수동으로 로그를 보지 않아도, **아래 한 가지만 실행**하면 진행 여부와 논문 대비 결과를 확인할 수 있습니다.

## 실행

```bash
cd dynamic_ensemble_rl_trading
python scripts/check_status_and_results.py
```

## 확인 내용

1. **진행 상태**
   - `미실행` / `진행중(학습)` / `진행중(백테스트 단계)` / `완료(백테스트만 실행됨)` / `완료(전체 파이프라인)`
   - 진행 중일 때: PPO 에이전트 수 (예: 5/15), 현재 풀·에이전트 (예: Bear 2/5)

2. **논문 대비 결과**
   - 평균 일치성(%)
   - 지표별: 논문 목표값, 실제값, 일치성(%)

결과는 **`results/verification/status_and_results.txt`** 에 저장됩니다.  
같은 스크립트를 다시 실행하면 최신 상태로 갱신됩니다.

## 참고

- 전체 파이프라인(학습 1M + 백테스트 + 비교)은 백그라운드에서 실행 중일 수 있습니다.  
  완료되면 상태가 `완료(전체 파이프라인)`으로 바뀌고, 결과가 갱신됩니다.
- 학습만 다시 하지 않고 백테스트·비교만 다시 하려면:  
  `python scripts/train_and_verify.py --backtest-only`
