# Reproducibility Statement (Reviewer #1/#2/#3, item #19)

This statement documents every artefact needed to reproduce the
results reported in ESWA-D-26-08980.

## 1. Code

* Public repository: https://github.com/heosanghun/2_ESWARegime
* Frozen commit / tag (to be filled at acceptance): `ESWA-final`
* Direct DOI (Zenodo mirror, to be created at acceptance): `10.5281/zenodo.<placeholder>`

The repository contains:

```
src/                      ← all production code
scripts/                  ← end-to-end pipelines
config/config.yaml        ← all experiment knobs
doc/                      ← manuscript revisions, this statement,
                            rebuttal letter, mapping tables
results/verification/     ← generated reports
```

## 2. Data

| Source | File | License |
|--------|------|---------|
| Binance Spot | `data/raw/btcusdt_1h.csv` (2021-10-12 → 2023-12-19) | Binance Public Data terms |
| CryptoNews aggregator | `data/cryptonews_2021-10-12_2023-12-19.csv` | author-collected, redistribution by reference |
| FinBERT-rescored news | `data/cryptonews_finbert_2021-10-12_2023-12-19.csv` | generated by `scripts/regenerate_news_sentiment_finbert.py` |

Raw data files are intentionally **not** stored in Git to keep the
repository small. They can be regenerated by:

```bash
python scripts/download_hourly_data.py        # OHLCV
python scripts/regenerate_news_sentiment_finbert.py
```

## 3. Random seeds

| Component | Seed |
|-----------|-----:|
| Numpy global seed | 42 |
| Python `random` | 42 |
| XGBoost regime classifier | 42 |
| PPO agents (5 per pool, Bull/Bear/Sideways) | 42 … 246 (per-agent offset of +1) |
| Stationary block bootstrap | 42 |
| Trend Scanning labeler | deterministic (no RNG) |

Seeds are set centrally by `src/utils/seed.py` and surfaced through
`scripts/train_and_verify.py:set_seed(...)`.

## 5. End-to-end reproduction commands

```bash
# 1. Install
git clone https://github.com/heosanghun/2_ESWARegime
cd 2_ESWARegime/dynamic_ensemble_rl_trading
python -m venv .venv && source .venv/bin/activate   # or .venv\Scripts\activate
pip install -r requirements.txt

# 2. Data
python scripts/download_hourly_data.py
python scripts/regenerate_news_sentiment_finbert.py

# 3. Train + verify (full)
python scripts/train_and_verify.py --reviewer3-mode

# 4. Reviewer #3 backtest-only verification
python scripts/train_and_verify.py --backtest-only --reviewer3-mode

# 5. Statistical robustness (item #9)
python scripts/run_statistical_tests.py

# 6. Computational complexity (item #12)
python scripts/measure_computational_complexity.py

# 7. News-removed ablation (item #13)
python scripts/run_ablation_no_news.py
```

Every script writes a deterministic, machine-readable artefact under
`results/verification/`, suitable for inclusion in supplementary
materials.
