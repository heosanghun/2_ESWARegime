# 로컬 데이터 전체 파악 및 박사논문 데이터 대조

**작성일:** 2026-02-13  
**목적:** 프로젝트에서 사용하는 모든 로컬 데이터를 나열하고, 박사논문에서 말하는 데이터와 동일한지 여부를 정리합니다.

---

## 1. 결론 요약

| 질문 | 답변 |
|------|------|
| **실제 박사논문에서 말하는 데이터로 테스트한 것인가?** | **코드베이스만으로는 논문이 명시한 정확한 데이터 출처·파일을 확인할 수 없습니다.** 다만, **기간·종목·봉 단위**는 스크립트 주석과 설정상 논문과 동일하게 맞춰 두었고, 현재 백테스트/검증은 **아래 로컬 데이터**로 수행됩니다. |

- 논문에서 사용한 **정확한 데이터셋(파일명, URL, 거래소 버전 등)** 은 논문 본문 또는 부록을 확인해야 합니다.
- 본 프로젝트는 **동일 기간·동일 종목(BTC/USDT 1시간봉)** 을 전제로 Binance에서 다운로드한 OHLCV와, 프로젝트 내 뉴스 CSV를 사용합니다.

---

## 2. 로컬 데이터 전체 목록

### 2.1 설정에서 참조하는 데이터 경로 (config/config.yaml)

| 설정 키 | 현재 값 | 용도 | 실제 사용 여부 |
|---------|----------|------|----------------|
| **data.ohlcv_path** | `data/raw/btcusdt_1h.csv` | OHLCV 1시간봉 (가격·거래량) | ✅ 백테스트·학습·검증에서 사용 |
| **data.news_path** | `data/cryptonews_2021-10-12_2023-12-19.csv` | 뉴스 감성 데이터 | ✅ FeatureFusion·백테스트에서 사용 |
| **data.processed_path** | `data/processed/` | 전처리 결과 저장 디렉터리 | 스크립트에 따라 생성·사용 가능 |
| **data.chart_images_path** | `D:/AI/TradingAgents/0_data/candlestick_images/chart_(7.42GB)` | 외부 캔들스틱 이미지 폴더 | ⚠️ **train_and_verify 백테스트에서는 미사용** (캔들은 OHLCV로 메모리에서 생성) |

### 2.2 프로젝트 내 실제 파일 (data/ 디렉터리)

| 파일 경로 | 설명 | 기간/형식 |
|-----------|------|-----------|
| **data/raw/btcusdt_1h.csv** | BTC/USDT 1시간봉 OHLCV (메인 사용) | 2021-10-12 00:00 ~ 2023-12-19 23:00, 컬럼: timestamp, open, high, low, close, volume |
| **data/raw/ohlcv_data.csv** | OHLCV 백업/대체용 (동일 또는 유사 데이터일 수 있음) | 설정에서 미지정 시 사용처 없음 |
| **data/cryptonews_2021-10-12_2023-12-19.csv** | 암호화폐 뉴스 (날짜, 감성, 제목, 본문 등) | 2021-10-12 ~ 2023-12-19, 약 31,014건 |

### 2.3 학습/테스트 기간 (config – training)

| 항목 | 설정 키 | 값 |
|------|----------|-----|
| 훈련 시작 | train_start_date | 2021-10-12 |
| 훈련 종료(테스트 시작) | train_end_date / test_start_date | 2023-06-19 |
| 테스트 종료 | test_end_date | 2023-12-19 |

즉, **백테스트에 쓰이는 구간**은 **2023-06-19 ~ 2023-12-19** 이고, 이 구간의 OHLCV는 `data/raw/btcusdt_1h.csv`에서 로드됩니다.

## 3. 데이터가 로드되는 코드 위치

| 데이터 | 로드 위치 | 비고 |
|--------|-----------|------|
| **OHLCV** | `src/data/data_processor.py` → `MarketDataHandler.load_data()`, `cfg['data']['ohlcv_path']` | `train_and_verify.py` Step3에서 test_start_date ~ test_end_date로 필터 |
| **뉴스** | `src/data/news_sentiment.py` → `NewsSentimentExtractor(news_path)`, `cfg['data']['news_path']` | `train_and_verify.py` Step3에서 동일 기간 필터 |
| **캔들 이미지** | `src/data/candlestick_generator.py` → OHLCV로부터 **메모리에서 생성** (CandlestickGenerator) | `chart_images_path`는 검증/체크 스크립트용, 메인 파이프라인에서는 미사용 |

## 4. OHLCV 데이터 출처 (다운로드 스크립트)

- **스크립트:** `scripts/download_hourly_data.py`
- **주석:** "Period: 2021-10-12 to 2023-12-19 (논문과 동일)"
- **출처:** **Binance** (ccxt), 심볼 **BTC/USDT**, 타임프레임 **1h**
- **저장 파일:** `data/raw/btcusdt_1h.csv`

즉, **논문과 동일한 기간**을 전제로 Binance에서 받은 데이터를 로컬 CSV로 쓰고 있으며, 논문이 “Binance BTC/USDT 1h”를 명시했는지는 **논문 원문 확인**이 필요합니다.

---

## 5. 뉴스 데이터

- **파일:** `data/cryptonews_2021-10-12_2023-12-19.csv`
- **컬럼 예:** date, sentiment, sentiment_polarity, sentiment_subjectivity, source, subject, title, text, url
- **기간:** 2021-10-12 ~ 2023-12-19 (OHLCV와 맞춤)
- 논문에서 **어떤 뉴스 데이터셋(출처·수집 방법)** 을 썼는지는 코드베이스에 없으므로, **동일 데이터로 테스트했다고 단정할 수 없습니다.**

---

## 6. chart_images_path (7.42GB 폴더)

- **경로:** `D:/AI/TradingAgents/0_data/candlestick_images/chart_(7.42GB)` (절대 경로)
- **역할:** 설정에만 있고, **메인 백테스트/학습 파이프라인(`train_and_verify.py`)에서는 사용하지 않음.**  
  캔들 시각 특징은 `CandlestickGenerator`가 **OHLCV만으로 메모리에서 생성**합니다.
- **사용처:** `scripts/check_data_requirements.py`, `scripts/comprehensive_verification.py` 등에서 존재 여부 확인용으로만 참조됩니다.

## 7. 박사논문 데이터와의 대조 체크리스트

| 항목 | 논문에서 확인 필요 | 현재 코드/로컬 데이터 |
|------|---------------------|---------------------------|
| 거래소 | 논문에 명시된 거래소 | Binance (download 스크립트) |
| 종목 | 논문에 명시된 심볼 | BTC/USDT |
| 봉 단위 | 논문에 명시된 타임프레임 | 1시간봉 |
| 기간(전체) | 논문 실험 기간 | 2021-10-12 ~ 2023-12-19 (스크립트·설정과 일치) |
| 테스트 기간 | 논문의 테스트 구간 | 2023-06-19 ~ 2023-12-19 (config와 일치) |
| OHLCV 파일/URL | 논문 또는 부록의 데이터 출처 | 로컬 `data/raw/btcusdt_1h.csv` (Binance 다운로드) |
| 뉴스 데이터셋 | 논문에 명시된 뉴스 출처·기간 | 로컬 `data/cryptonews_2021-10-12_2023-12-19.csv` (동일 여부 미확인) |
| 캔들 이미지 | 논문이 전처리 이미지 사용 여부 | 코드는 OHLCV 기반 생성만 사용, 외부 7.42GB 폴더는 미사용 |

## 8. 요약 표: 로컬 데이터 vs 논문

| 구분 | 로컬 데이터 | 논문과 동일한지 |
|------|-------------|------------------|
| **OHLCV** | `data/raw/btcusdt_1h.csv` (Binance BTC/USDT 1h, 2021-10-12~2023-12-19) | 기간·종목·봉 단위는 “논문과 동일”로 맞춤. **정확한 일치 여부는 논문 원문 확인 필요.** |
| **뉴스** | `data/cryptonews_2021-10-12_2023-12-19.csv` | 출처·구성이 논문과 동일한지 **코드베이스로는 확인 불가.** |
| **캔들 이미지** | 메모리 생성 (OHLCV 기반) / 외부 `chart_images_path`는 미사용 | 논문이 별도 이미지 데이터를 썼다면 **다를 수 있음.** |
| **테스트 기간** | 2023-06-19 ~ 2023-12-19 | 설정상 논문과 동일하게 맞춰 둔 상태. |

**정리:**  
- **로컬 데이터는 모두 위 목록이 전부**이며, 백테스트/성과 검증은 이 로컬 OHLCV·뉴스와 설정 기간으로 수행됩니다.  
- **“실제 박사논문에서 말하는 데이터로 테스트한 것인가?”**에 대한 확정 답은 **논문에서 사용한 데이터셋을 명시한 부분을 확인**한 뒤, 위 표의 항목과 비교해야 합니다.
